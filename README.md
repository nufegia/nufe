# Nufe

**Nufe** 是一门正在建设中的人工语言。[点此进入网站](https://nufe-language.vercel.app)。

这门语言并不是在模仿某一种自然语言，而是在尝试建立一套更整齐、更可整理、也更适合持续扩展的表达系统。它目前已经有比较成形的语法说明、词汇资料，以及一批用于世界、存在、关系、行动与专名表达的基础词汇。

Nufe 的特点主要在于：

- 风格简洁，规则意识很强
- 词汇设计带有明显的抽象表达取向
- 不只关注日常表达，也关注世界观、存在论和概念表达
- 已经开始认真处理外来词、国名、地名和文明传统名称的引入方式

这个仓库目前收录的是 Nufe 的核心资料，包括：

- 语法文档
- 词库
- 特殊词与数词资料
- 一个用于浏览和学习这些内容的网页界面

如果你第一次接触 Nufe，可以把它理解为一个还在生长中的语言项目：它已经有了清晰的骨架，也已经显露出自己的气质，但仍然保留着继续扩展和定型的空间。
