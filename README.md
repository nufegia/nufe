# Nufe

**Nufe** 是一门正在建设中的人工语言。[点此进入网站](https://nufe-language.vercel.app)。

它并不是为了模仿某一种自然语言而被设计出来的。Nufe 更像是在尝试建立一套结构更整齐、规则更清楚、便于整理与扩展的表达系统。现在，这门语言已经有了较为成形的语法说明、词汇资料，以及一批围绕世界、存在、关系、行动与专名表达建立起来的基础词汇。

Nufe 目前的特点主要包括：

- 风格简洁，规则意识明确
- 词汇设计带有明显的抽象表达取向
- 不只关注日常交流，也关注世界观、存在论与概念表达
- 已经开始系统处理外来词、国名、地名和文明传统名称的引入方式

Nufe 的一个重要意图，是让语言不仅适合人学习和使用，也更适合被 AI 理解、整理与生成。这里所说的“支持 AI”，并不是把语言做成机器代码，而是希望它在表达层面尽量减少不必要的歧义、让规则更稳定、让词汇与语法更容易被系统化描述。对 AI 来说，这意味着它更容易被分词、标注、建模、扩写、翻译和持续维护；对人来说，这意味着语言资料可以更一致地积累，也更容易形成可检索、可训练、可迭代的语料基础。

因此，Nufe 的设计并不只是在追求“好不好看”或“像不像自然语言”，也在认真考虑一个问题：如果未来越来越多的语言活动都会和 AI 协作发生，那么能否存在一种语言，从一开始就把“人可用”与“AI 可处理”同时纳入设计视野。Nufe 想探索的，正是这样的方向。

这个仓库目前收录的是 Nufe 的核心资料，包括：

- 语法文档
- 词库
- 特殊词与数词资料
- 一个用于浏览和学习这些内容的网页界面

如果你是第一次接触 Nufe，可以把它理解为一个仍在生长中的语言项目：它已经有了清晰的骨架，也逐渐显露出自己的气质，但仍然保留着继续扩展、修正与定型的空间。
