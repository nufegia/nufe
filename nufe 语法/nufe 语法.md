# 《Nufe 语法》

**日期：** 2026-04-09

[TOC]

## 一、字音

Nufe 使用 5 个单元音、8 个双元音，以及相应的鼻音形式。双元音的结构为“介音 + 元音”。

### 1. 单元音与双元音

| 类型   | 形式                                                         |
| ------ | ------------------------------------------------------------ |
| 单元音 | `a` [a]，`i` [i]，`u` [u]，`e` [ɛ]，`o` [o]                  |
| 双元音 | `ia` [ja]，`ie` [jɛ]，`io` [jo]，`iu` [ju]，`ua` [wa]，`ue` [wɛ]，`uo` [wo]，`ui` [wi] |

### 2. 鼻音形式

Nufe 设有一个鼻音音素 `-w` [ŋ/n/m]，可附着在单元音或双元音之后，用以构成鼻化元音。

单元音的鼻音形式为：`aw` [aŋ/an/am]，`iw` [iŋ/in/im]，`uw` [uŋ/un/um]，`ew` [ɛŋ/ɛn/ɛm]，`ow` [oŋ/on/om]。  
双元音的鼻音形式为：`iaw` [jaŋ/jan/jam]，`iew` [jɛŋ/jɛn/jɛm]，`iow` [joŋ/jon/jom]，`iuw` [juŋ/jun/jum]，`uaw` [waŋ/wan/wam]，`uew` [wɛŋ/wɛn/wɛm]，`uow` [woŋ/won/wom]，`uiw` [wiŋ/win/wim]。

### 3. 辅音

Nufe 使用 18 个单辅音，可分为以下五组：

| 类型         | 形式                                         |
| ------------ | -------------------------------------------- |
| 唇音         | `p` [pʰ]，`b` [b]，`m` [m]，`f` [f]，`v` [v] |
| 舌音         | `t` [tʰ]，`d` [d]，`n` [n]，`s` [s]，`z` [z] |
| 软腭音与喉音 | `k` [kʰ]，`g` [g]，`q` [ŋ]，`h` [h]，`r` [ɦ] |
| 塞擦音       | `c` [tʃʰ]，`j` [dʒ]                          |
| 边音         | `l` [l]                                      |

## 二、音节

Nufe 的音节结构为 **(C)(G)V(N)**，即“辅音—介音—元音—鼻音”。其中，辅音、介音与鼻音都可以省略，但元音不可省略。

若音节不带辅音，则书写时需在前加字母 `y`。例如：`ya` [a]，`yi` [i]，`yu` [u]，`ye` [ɛ]，`yo` [o]。

全语言的合法音节共 494 个，例如：`ya`、`ki`、`lui`、`me`、`po`、`faw`、`dew`、`muo`、`ci`、`tow`。

## 三、词

词由一个或多个音节组成，例如：`sa`、`patuwmi`、`kilaw`、`somitucano`。

若词首音节无辅音，书写时可省略词首字母 `y`。

词内音节遵循以下调式：最末音节读低降调；倒数第二个音节读高平调，并构成整个词最明亮、最易辨认的部分；倒数第二个音节之前的其他音节读低平调。由此，在口语中，词与词之间的界限较为清晰；在书写中，词与词之间以空格分隔。

词与词还可以拼合构成新词，例如：`haho` + `hi` = `hahohi`。

## 四、词类

Nufe 的词按照句法功能与意义，可分为实体、动作、性质和功能四类。

### 1. 实体

实体用于指称人、事物、地点、概念等具体或抽象的对象，也包括代词。在句中通常充当主语或宾语。

### 2. 动作

动作用于表示实体发出的动态行为，或实体所处的静态状态。在句中充当谓语。

按照是否需要宾语，动作可进一步区分如下：

- **无对象动作**：不需要宾语即可完整表达意义，通常表示主语自身发出的动作或所处状态，如“跑”“睡”“笑”“存在”。
- **有对象动作**：需要一个直接宾语来完整表达意义，宾语通常是动作的承受者或对象，如“吃”“看”“写”“建造”。
- 某些有对象动作还需要两个宾语，通常包括一个直接宾语和一个间接宾语；其中，间接宾语多表示受益者、接受者或目的地，如“给”“教”“告诉”“寄送”。

### 3. 性质

性质用于描述实体、动作或性质本身的属性、特征、数量或序数。在句中作修饰语。

### 4. 功能

功能词本身不承担独立实义，而承担语法作用。它们可用于连接词、短语或句子，也可用于表达说话者的态度、情绪或意图，如疑问、肯定、感叹、命令、请求等。

### 5. 定义原则

一个词可以有多个意义或用法，但只能归入一种词类。

词条可依数据表格的方式定义，例如：

| 词   | 类   | 翻译                 | 描述                                                     | 例句 |
| ---- | ---- | -------------------- | -------------------------------------------------------- | ---- |
| nufe | 实体 | 语言、方言；Nufe语言 | 泛指所有语言，即人类沟通的符号系统。也用于特指这门语言。 |      |

## 五、词类转化

实体、动作和性质之间可以发生转化。

实体可以转化为性质，即一个表示人、事、物的词，也可以用来描述另一个事物的特征。例如，“石头”可以转化为“石头的”，如“石头墙”。

性质可以转化为实体，即一个表示特征的词，也可以转化为表示抽象概念或具体事物的词。例如，“美丽”可以转化为“美”，如“追求美”。

动作也可以转化为实体，即一个表示行为的词，也可以转化为表示该行为本身或相关概念的词。例如，“跑”可以转化为“跑步”，如“跑步是他的爱好”。

转化的书写方式是在词末添加字母 `x`。其发音方式如下：`-ax` [-aə]，`-ix` [-iə]，`-ux` [-uə]，`-ex` [-ɛə]，`-ox` [-oə]，`-wx` [-nnə / -mmə / -ŋŋə]。

## 六、26 个功能词

26 个无辅音音节词先定为功能词。它们是：

`a`、`i`、`u`、`e`、`o`；`ia`、`ie`、`io`、`iu`、`ua`、`ue`、`uo`、`ui`；`aw`、`iw`、`uw`、`ew`、`ow`；`iaw`、`iew`、`iow`、`iuw`、`uaw`、`uew`、`uow`、`uiw`。

## 七、动作的时态与体态

动作词的时态与体态，通过置于动作词前的功能词表达。

### 1. 时态功能词

时态用于表示动作发生的时间：

- `aw`：现在时，用于描述当前时间范围内发生的活动状态。
- `iw`：过去时，用于标记动作发生在说话时间之前。
- `uw`：将来时，用于标记动作发生在说话时间之后。

### 2. 体态功能词

体态用于表示动作的内部状态：

- `ew`：完成体，用于标记动作已经完成，强调其结果或终结性。
- `ow`：进行体，用于标记动作正在进行，强调其持续性或过程性。

### 3. 基本语序

基本语序为：

`[时态功能词] + [体态功能词] + 动作词`

动作词也可以不带时态或体态功能词，其意义由上下文决定，通常可表示一般事实或习惯性动作。

### 4. 示例

例如，`我 吃` 可依语境理解为“我吃”“我在吃”或“我会吃”。

若只标记时态，则：

- `我 aw 吃`：我现在吃。
- `我 iw 吃`：我曾吃。
- `我 uw 吃`：我会吃。

若只标记体态，则：

- `我 ew 吃`：我已经吃完了。
- `我 ow 吃`：我正在吃。

时态与体态还可以组合，形成六种常用形式：

| 组合          | 名称       | 示例        | 含义             |
| ------------- | ---------- | ----------- | ---------------- |
| `aw ew = iaw` | 现在完成体 | `我 iaw 吃` | 我现在已经吃完了 |
| `aw ow = uaw` | 现在进行体 | `我 uaw 吃` | 我现在正在吃     |
| `iw ew = iew` | 过去完成体 | `我 iew 吃` | 我当时已经吃完了 |
| `iw ow = iow` | 过去进行体 | `我 iow 吃` | 我当时正在吃     |
| `uw ew = uew` | 将来完成体 | `我 uew 吃` | 我将来会吃完     |
| `uw ow = uow` | 将来进行体 | `我 uow 吃` | 我将来会正在吃   |

## 八、修饰

Nufe 中存在以下几种修饰关系。

### 1. 实体修饰实体

实体可以修饰实体，表示两者之间的某种关联关系。这种关系可以是所属或拥有关系，如“人的手”“妈妈的女儿”；也可以是部分与整体关系，如“车的轮子”；还可以表示种类、类型或来源，如“苹果树”“石头墙”“北京烤鸭”；也可以表示用途或目的，如“咖啡杯”；此外，还包括指示关系，如“这本书”“那些花”，以及计数或排序关系，如“三个苹果”“第一个人”。

### 2. 性质修饰实体

性质可以修饰实体，用于描述或限定实体的属性或特征，如“红色的球”“高大的建筑”。

### 3. 性质修饰动作

性质可以修饰动作，用于描述动作发生的方式或程度，如“快速地跑”“慢慢地吃”。

### 4. 性质修饰性质

性质还可以修饰性质，通常表示程度，如“非常红”“极其快速地跑”。

### 5. 动作修饰动作

动作也可以修饰动作，通常表示意愿、能力或方式等，如“想吃”“能做”。

## 九、句子

句子的基本成分包括主语、谓语和宾语。主语是动作或状态的发出者、承担者；谓语是对主语的动作或状态所作的表述；宾语是部分动作词所要求的对象或承受者。

### 1. 基本语序

句子的核心语序为：

`主语 + 谓语`

如果动作词属于有对象动作，则其后可以接宾语，即：

`主语 谓语 [宾语]`

### 2. 同类成分并列

当句中出现多个同类成分，如多个主语、谓语或宾语时，可以使用功能连词 `e` 将其并列连接。例如：

`主语 e 主语 谓语 e 谓语 宾语 e 宾语`

### 3. 修饰原则

句中修饰遵循以下三项原则：

1. **前置修饰**：修饰语置于被修饰成分之前。
2. **相邻原则**：修饰语直接修饰其后紧邻的成分。
3. **同类并列**：多个同类、同级的修饰语可以使用功能连词 `e` 连接，以表示并列修饰。

例如：

`非常 红 e 绿 球`

表示“非常红的、绿的球”。

### 4. 句式概括

句子的基本模式可以概括为：

`[修饰语 e 修饰语] 主语 e 主语 [修饰语 e 修饰语] 谓语 e 谓语 [修饰语 e 修饰语] 宾语 e 宾语`

> 注意：性质词不能作谓语，只有动作词可以作谓语。

## 十、句子转化

某些动作词，如 `说`、`认为`，要求以整个句子作为主语或宾语。因此，Nufe 允许将句子结构转化为实体形式。

### 1. 句子转化为实体

将句子转化为实体时，使用结构：

`i [句子] u`

在没有语法歧义、且前后存在停顿时，`i` 或 `u` 可以省略其一。

### 2. 句子转化为修饰语

此外，还可以使用功能词 `o` 将句子转化为后置修饰语，用来修饰其前面的成分。其结构为：

`[被修饰成分] o [句子]`

在该结构中，句子部分之后应有停顿；书写时可用 `,` 或 `.` 表示。这样一来，动作便能够以句子的形式参与修饰。例如，“读”可转化为“在读的”，如“在读的人”。

## 十一、性质判断句

表达性质判断时，使用如下模式：

`实体词 + “是” + 性质词 + “东西”`

## 十二、其他介词及连接词

Nufe 还使用若干特殊介词和连接结构。

### 1. 特殊介词

- `ia`：表示“在……”或“给……”，可用于引入间接宾语。
- `ie`：表示“从……”或“通过……”，分别对应来源与方式。
- `ue`：表示“去……”或“为了……”，分别对应去向与目的。
- `ua`：表示“或、或者”。

### 2. 连接结构

- `io ... uo`：虽然……但是……
- `iu ... ui`：如果 / 若……那么 / 则……
- `iuw ... uiw`：因为……所以……

## 十三、语气

功能词 `a` 用来表达语气。它可以出现在句首、句中或句末，用于表达说话者的情感、态度或语气色彩。陈述、疑问、感叹、祈使等不同语气，都可以通过 `a` 的使用体现。

例如，在句末加上 `a`，句子可以带有提示、强调、感叹、祈使或疑问等语气。其具体含义由上下文、语调和语境共同决定。

## 十四、提问

一般疑问句，即是 / 否问题，在句末使用功能词 `a?`。

特殊疑问句通过用疑问词替换所提问的成分来构成。其中，用 `ha` 替换实体词，用于提问“什么事物”。

## 十五、数的规则

Nufe 的数根如下：

| 数值 | 形式 |
| ---- | ---- |
| 1    | `pa` |
| 2    | `fa` |
| 3    | `ma` |
| 4    | `ta` |
| 5    | `sa` |
| 6    | `na` |
| 7    | `ka` |
| 8    | `ha` |
| 9    | `qa` |
| 0    | `la` |

此外，`lia` 表示百，`lua` 表示千。

数根拼合后形成数的基本形态，称为**数元**。数元不是词。例如，123 写作 `pafama`。

`cu` 表示小数点，因此 3.14 写作 `macupata`。  
`ci` 表示分数，因此 3/14 写作 `macipata`。

在数元后加后缀 `ce`，可形成基数词，用于计数，例如：`pace`“一个”，`face`“二 / 两个”，`paluace`“1000个”，`macupatace`“3.14个”，`macipatace`“14分之3个”。

在数元后加后缀 `co`，可形成序数词，例如：`paco`“第一”，`faco`“第二”，`mahafapaco`“第三千八百二十一”，`paliaco`“第一百”，`paluaco`“第一千”。

## 十六、外来词引入

借用外来词时，需要将其发音同化为合法音节。对于难以简化的复辅音，应插入元音 `a` 进行拆分，例如 `ksi` 变为 `kasi`。

外来词通常作为实体词引入。

引入外来词时，以**来源语言的标准现代读音**或**约定俗成的古典读音**为准，而不机械照搬原文字母。专名、地名、国名、族名、机构名、历史名词、宗教名词，原则上均应先确定其来源语言，再进行音节同化。

| 伊斯兰 | 古典阿拉伯语 | 《古兰经》、古典教法、神学、哲学与传统宗教专名。 |
| 犹太 | 圣经希伯来语 | 《希伯来圣经》、古代犹太宗教与相关专名。 |
| 两河 | 苏美尔语 / 阿卡德语 | 苏美尔、巴比伦、亚述及相关古代专名；按史实所属语言分别处理。 |
| 埃及 | 古埃及语 | 法老时代王名、神名、地名、宗教与制度专名。 |

同一文化若同时存在古典层与现代层，则应区分处理：

- 古代人物、古代文献、古代制度、神话宗教专名，依古典语言引入。
- 近现代人物、近现代国家机构、现代城市与现代社会事物，依现代国家与政治体部分所列语言引入。

### 3. 引入总则

外来词引入 Nufe 时，一律先确定**来源语言的实际读音**，再按下列 IPA 到 Nufe 的规则映射。不得直接按外语拼写逐字母转写。

#### 3.1 Nufe 可接收的音系形式

Nufe 的外来词最终必须落入本语合法音节 `(C)(G)V(N)`。其中：

- `C` 为辅音：`p b m f v t d n s z k g q h r c j l`
- `G` 为介音：`i` 或 `u`
- `V` 为元音：`a i u e o`
- `N` 为鼻尾：`-w`

因此，任何外来音都必须先映射到上述系统，再重组为合法音节。

#### 3.2 IPA 到 Nufe 的基本映射

##### 1. 辅音

下表为 IPA 到 Nufe 的辅音映射。带有位置说明者，按所列位置条件唯一执行。

| IPA | Nufe | 说明 |
| --- | --- | --- |
| `/p b m f v/` | `p b m f v` | 直接对应 |
| `/t d n l/` | `t d n l` | 直接对应 |
| `/k g/` | `k g` | 直接对应 |
| `/ŋ/` | `q` / `-w` | 音节首一律作 `q`，音节尾一律作 `-w` |
| `/h/` | `h` | 直接对应 |
| `/r ɾ ɹ/` | `l` | 各类流音性 r 默认并入 `l` |
| `/ʁ ʀ/` | `r` | 喉化、小舌化的 r 音并入 `r` |
| `/s z/` | `s z` | 直接对应 |
| `/ts tsʰ/` | `c` | 齿龈清塞擦音统一并入 `c` |
| `/dz/` | `j` | 齿龈浊塞擦音统一并入 `j` |
| `/ʃ ʂ/` | `s` | 龈后清擦音统一并入 `s` |
| `/ʒ ʐ/` | `z` | 统一并入 `z` |
| `/tʃ tɕ/` | `c` | 清塞擦音 |
| `/dʒ dʑ/` | `j` | 浊塞擦音 |
| `/θ ð/` | `s z` | 英语、希腊语等齿擦音统一并入 |
| `/x χ ħ ḥ ḫ/` | `h` | 喉擦音、小舌擦音统一并入 |
| `/ɣ/` | `g` | 浊软腭擦音默认作 `g` |
| `/q/` | `k` | 小舌塞音不保留 |
| `/ɲ/` | `ni` | 腭鼻音展开为 `n + i` |
| `/ʎ/` | `li` | 腭边音展开为 `l + i` |
| `/j/` | `i` 介音 | 与后元音结合成 `ia ie io iu` |
| `/w/` | `u` 介音 | 与后元音结合成 `ua ue uo ui` |

##### 2. 元音

| IPA | Nufe | 说明 |
| --- | --- | --- |
| `/a ɑ ɐ/` | `a` | 低元音并入 `a` |
| `/i ɪ y ʏ/` | `i` | 前高元音与前圆唇高元音并入 `i` |
| `/u ʊ/` | `u` | 后高元音并入 `u` |
| `/e ɛ ə ɘ ɤ ø œ/` | `e` | 中前、中中央与前圆唇中元音并入 `e` |
| `/o ɔ ɵ/` | `o` | 中后与后圆唇元音并入 `o` |
| 鼻化元音 | 对应元音 + `-w` | 如 `/ã/ → aw`，`/õ/ → ow` |

##### 3. 双元音与复元音

Nufe 只直接保留八种介音结构：`ia ie io iu ua ue uo ui`。其他复元音一律拆分：

| IPA | Nufe |
| --- | --- |
| `/ai/` | `ayi` |
| `/ei/` | `eyi` |
| `/oi/` | `oyi` |
| `/au/` | `ayu` |
| `/ou/` | `oyu` |
| `/ao/` | `ayo` |
| `/ia/ /ja/` | `ia` |
| `/ie/ /je/` | `ie` |
| `/io/ /jo/` | `io` |
| `/iu/ /ju/` | `iu` |
| `/ua/ /wa/` | `ua` |
| `/ue/ /we/` | `ue` |
| `/uo/ /wo/` | `uo` |
| `/ui/ /wi/` | `ui` |

#### 3.3 音节调整规则

1. 复辅音不保留，必须拆开。默认插入 `a`，如无更强语音依据，不用别的插音。
2. 词尾 `-m -n -ng` 一律写作 `-w`。
3. 词尾其他辅音一律补元音开音节化，默认补 `a`。
4. 长元音、重音、声调、送气、紧音、清浊、软硬、强调等区别一律不进入 Nufe 词形；只保留其较稳定的音质骨架。
5. 若某语言已有约定俗成的学术读音或传统读音，则可先依该读音，再按本节映射。
6. 对现代专名，若严格逐音节映射所得词形过长，而较短形式已在本语言内部稳定使用，则可采用**专名约化**。专名约化只可做两类处理：一是将复元音并入其较稳定的主元音， `ai ei` 可并作 `e`，`ou` 可并作 `o`；二是将相邻擦音、塞擦音并入 Nufe 中更稳定、辨识度更高的近似音位。约化后词形仍须满足合法音节 `(C)(G)V(N)`。

### 4. 核心现代语言

以下语言是现代借词进入 Nufe 的核心来源语言。每种语言都须先按本节规则定音，再执行总则。

#### 4.1 英语

英语以现代标准读音为准，不按拼写转写。

- 辅音：
  - `/ʃ/ → s`
  - `/ʒ/ → z`
  - `/tʃ/ → c`
  - `/dʒ/ → j`
  - `/θ/ → s`
  - `/ð/ → z`
  - `/r/ → l`
  - `/w/` 与后元音合写为 `ua ue uo ui`
  - `/j/` 与后元音合写为 `ia ie io iu`
- 元音：
  - `/i iː ɪ/ → i`
  - `/e ɛ eɪ/ → e`
  - `/æ ʌ ə ɑ/ → a`
  - `/o ɒ ɔ oʊ/ → o`
  - `/u ʊ uː/ → u`
  - `/aɪ aʊ ɔɪ/ → ayi ayu oyi`
- 词形：
  - 词首复辅音一律拆开：`st- → sata-`，`sp- → sapa-`，`kr- → kara-`
  - 词尾辅音补 `a`，鼻尾改 `-w`

#### 4.2 现代标准汉语

现代标准汉语以普通话读音为准，不记声调。

- 声母：
  - `b p m f d t n l g k h` 保持不变
  - `j q x → j c s`
  - `zh ch sh → c c s`
  - `z c s → c c s`
  - `r → z`
- 韵母：
  - `a o e i u → a o e i u`
  - `ia ie io iu ua ue uo ui` 保持不变
  - `ai ei ao ou → ayi eyi ayo oyu`
  - `an ang → aw`
  - `en eng → ew`
  - `in ing → iw`
  - `ong → ow`
  - `iong → iow`
  - `un → uw`
  - `ün → iuw`
  - `iao → iayo`
- 特别规则：
  - 儿化不保留
  - `zhi chi shi ri zi ci si` 一律作带 `i` 的音节

#### 4.3 法语

法语以现代标准法语读音为准，不按正字法。

- 辅音：
  - `/ʃ/ → s`
  - `/ʒ/ → z`
  - `/ɲ/ → ni`
  - `/ʁ/ → r`
  - `/j/` 一律作 `i` 介音
  - `/ɥ/` 统一作 `ue`
  - `/w/` 一律作 `u` 介音
- 元音：
  - `/i y/ → i`
  - `/u/ → u`
  - `/e ɛ ø œ ə/ → e`
  - `/a/ → a`
  - `/o ɔ/ → o`
  - `/ɑ̃/ → aw`
  - `/ɛ̃ œ̃/ → ew`
  - `/ɔ̃/ → ow`
- 词形：
  - 不发音词尾不写
  - 发音词尾辅音补 `a`

#### 4.4 俄语

俄语以现代标准俄语读音为准，不记重音。

- 辅音：
  - `/x/ → h`
  - `/ʂ/ → s`
  - `/ʐ/ → z`
  - `/tɕ/ → c`
  - `/ɕː/ → s`
  - 软辅音不单列，一律用后续 `i/e` 近似
- 元音：
  - `/i e a o u/ → i e a o u`
  - 弱化元音仍按实际音色落到五元音
- 词形：
  - 复辅音拆开
  - 词尾鼻音改 `-w`
  - 词尾其他辅音补 `a`

#### 4.5 印地语

印地语以现代标准印地语口语为准，schwa 省落按实际发音处理。

- 辅音：
  - 送气不保留：`ph bh th dh kh gh → p b t d k g`
  - 卷舌 `ṭ ḍ ṇ → t d n`
  - `c ch → c`
  - `j jh → j`
  - `ś ṣ s → s`
  - `ṛ → li`
- 元音：
  - `a ā → a`
  - `i ī → i`
  - `u ū → u`
  - `e → e`
  - `o → o`
  - `ai → ayi`
  - `au → ayu`
  - 鼻化元音 → 对应元音加 `-w`
- 词形：
  - 脱落的 schwa 不恢复
  - 词尾辅音补 `a`

#### 4.6 现代标准日语

日语以现代共通语拍节读音为准。

- 辅音：
  - `sh → s`
  - `ch → c`
  - `ts → c`
  - `j → j`
  - `y` 并入 `ia ie io iu`
  - `w` 并入 `ua ue uo ui`
- 元音：
  - `a i u e o → a i u e o`
  - 长元音不保留
- 特殊规则：
  - `shi, chi, tsu, fu → si, ci, cu, fu`
  - 撥音 `N`：词尾作 `-w`，词中按邻近鼻音处理
  - 促音不保留

#### 4.7 标准德语

德语以标准德语读音为准。

- 辅音：
  - `/ʃ/ → s`
  - `/ʒ/ → z`
  - `/ç x/ → h`
  - `/ts/ → c`
  - `/pf/ → p`
  - `/ʁ ʀ/ → r`
  - `/r ɾ/ → l`
- 元音：
  - `/i ɪ/ → i`
  - `/e ɛ ø œ/ → e`
  - `/a/ → a`
  - `/o ɔ/ → o`
  - `/u ʊ/ → u`
  - `/y ʏ/ → i`
  - `/ai au ɔy/ → ayi ayu oyi`
- 词形：
  - 词尾清化不恢复拼写
  - 词尾辅音补 `a`

#### 4.8 标准韩语

韩语以现代首尔标准音为准。

- 辅音：
  - `ㅂ ㅃ ㅍ → p`
  - `ㄷ ㄸ ㅌ → t`
  - `ㄱ ㄲ ㅋ → k`
  - `ㅈ ㅉ → j`
  - `ㅊ → c`
  - `ㅅ ㅆ → s`
  - `ㄹ → l`
  - `ㅎ → h`
- 元音：
  - `ㅏ → a`
  - `ㅓ → e`
  - `ㅗ → o`
  - `ㅜ ㅡ → u`
  - `ㅣ → i`
  - `ㅐ ㅔ → e`
  - `ㅑ ㅕ ㅛ ㅠ → ia ie io iu`
  - `ㅘ ㅝ → ua ue`
  - `ㅚ ㅟ → ui`
- 词形：
  - 终声塞音不保留闭塞，补 `a`
  - 终声鼻音中，`ㅇ` 词尾作 `-w`

#### 4.9 意大利语

意大利语以标准意大利语读音为准。

- 辅音：
  - `/ʃ/ → s`
  - `/tʃ/ → c`
  - `/dʒ/ → j`
  - `/ɲ/ → ni`
  - `/ʎ/ → li`
- 元音：
  - `/a e ɛ i o ɔ u/ → a e e i o o u`
- 词形：
  - 双辅音简化
  - 词尾元音通常保留

#### 4.10 土耳其语

土耳其语以现代标准土耳其语读音为准。

- 辅音：
  - `ş → s`
  - `ç → c`
  - `c → j`
  - `y` 并入 `ia ie io iu`
  - `ğ` 不单写，只并入邻近元音
- 元音：
  - `a e i o u → a e i o u`
  - `ı → u`
  - `ö → e`
  - `ü → i`
- 词形：
  - 元音和谐不保留
  - 词尾辅音补 `a`

#### 4.11 西班牙语

西班牙语以现代标准读音为准。

- 辅音：
  - `/x/ → h`
  - `/tʃ/ → c`
  - `/ɲ/ → ni`
  - `/ʎ ʝ/ → j`
  - `/ɾ r/ → l`
  - `/θ/` 若存在，作 `s`
- 元音：
  - `/a e i o u/ → a e i o u`
- 词形：
  - 滑音按 `ia ie io iu ua ue uo ui`
  - 词尾 `-n -m → -w`
  - 词尾其他辅音补 `a`

#### 4.12 葡萄牙语

葡萄牙语以现代标准读音为准；巴西专名可依巴西读音。

- 辅音：
  - `/ʃ/ → s`
  - `/ʒ/ → z`
  - `/ɲ/ → ni`
  - `/ʎ/ → li`
  - `/ʁ ʀ/ → r`
  - `/r ɾ ɹ/ → l`
- 元音：
  - `/a e ɛ i o ɔ u/ → a e e i o o u`
  - 鼻化 `ã õ ẽ ĩ ũ → aw ow ew iw uw`
- 词形：
  - 词尾 `-m -n → -w`
  - 词尾擦音一律作 `s`，再补元音

#### 4.13 印度尼西亚语

印度尼西亚语基本按现代标准读音直转。

- 辅音：
  - `c → c`
  - `j → j`
  - `sy → s`
  - `ny → ni`
  - `ng`：词尾作 `-w`，词首或词中音节首作 `q`
- 元音：
  - `a i u e o → a i u e o`
  - 央化 `e` 仍作 `e`
- 词形：
  - 外来复辅音拆开
  - 词尾辅音补 `a`

#### 4.14 现代标准阿拉伯语

现代标准阿拉伯语以通行朗读音为准。

- 辅音：
  - `ث ذ → s z`
  - `ش → s`
  - `ج → j`
  - `خ ح → h`
  - `غ → g`
  - `ق → k`
  - `ع` 一律不写
  - `ص ض ط ظ → s d t z`
  - `و ي` 分别并入 `u` 介音与 `i` 介音
- 元音：
  - `a i u ā ī ū → a i u`
  - `ay aw → ayi ayu`
- 词形：
  - 格尾不写
  - 定冠词同化按实际读音
  - 词尾辅音补 `a`

### 5. 古典语言

古典语言一律按本节规定的标准学术读音转写，不并用其他重建体系。

#### 5.1 梵语

- 辅音：
  - 送气不保留：`kh gh ch jh ṭh ḍh th dh ph bh → k g c j t d t d p b`
  - 卷舌 `ṭ ḍ ṇ ṣ → t d n s`
  - `ś ṣ s → s`
  - `jñ → jani`
  - `ṛ ḷ → li li`
- 元音：
  - `a ā → a`
  - `i ī → i`
  - `u ū → u`
  - `e o → e o`
  - `ai au → ayi ayu`
  - `ṃ ṃ̐` 鼻化作 `-w`
- 词形：
  - 词尾辅音补 `a`

#### 5.2 古希腊语

- 辅音：
  - `ph th kh → p t k`
  - `ch` 类后期擦音一律作 `k`
  - 粗气号作 `h`
  - `r` 作 `l`
- 元音：
  - 长短不保留
  - `a e ē i o ō u → a e e i o o u`
  - `y → i`
  - `ai ei oi au eu ou → ayi eyi oyi ayu eyu oyu`
- 词形：
  - 词尾辅音补 `a`

#### 5.3 拉丁语

- 辅音：
  - `c` 恒作 `k`
  - `g` 恒作 `g`
  - `qu → kua`
  - `v` 一律作 `v`
- 元音：
  - 长短不保留
  - `a e i o u → a e i o u`
  - `ae oe au → ayi oyi ayu`
- 词形：
  - 词尾 `-m → -w`
  - 词尾其他辅音补 `a`

#### 5.4 中古汉语

中古汉语一律采用本节规定的工作性重建方案，并只保留可稳定转入 Nufe 的音系区别。

- 声母：
  - 唇、舌、牙喉诸塞音按 `p b m f / t d n l / k g h` 归并
  - 知庄章组塞擦音与擦音按 `c j s` 归并
  - 日母默认作 `z`
- 韵母：
  - 开合口分别按普通的单元音、介音结构归入 `a i u e o` 与 `ia ie io iu ua ue uo ui`
  - `-m -n -ng → -w`
  - 入声 `-p -t -k` 不保留闭尾，分别开化为 `-pa -ta -ka`
- 词形：
  - 声调不记

#### 5.5 古典阿拉伯语

- 辅音：
  - `ث ذ → s z`
  - `ش → s`
  - `ح خ → h`
  - `غ → g`
  - `ق → k`
  - `ع` 一律省略
  - `ص ض ط ظ → s d t z`
- 元音：
  - `a i u ā ī ū → a i u`
  - `ay aw → ayi ayu`
- 词形：
  - 格尾通常不写
  - 词尾辅音补 `a`

#### 5.6 圣经希伯来语

- 辅音：
  - `š → s`
  - `ṣ → s`
  - `ḥ ḵ h → h`
  - `ʿ` 一律省略
  - `r → l`
- 元音：
  - 统一归入 `a i u e o`
  - 历史长短不保留
- 词形：
  - 词尾辅音补 `a`

#### 5.7 苏美尔语

- 辅音：
  - `p b t d k g m n s l r → p b t d k g m n s l l`
  - `ŋ` 词首或词中作 `q`，词尾作 `-w`
- 元音：
  - `a e i u` 直转
- 词形：
  - 词尾辅音补 `a`

#### 5.8 阿卡德语

- 辅音：
  - `š → s`
  - `ṣ ṭ q → s t k`
  - 其他塞音、鼻音、流音按最近值直转，其中 `r → l`
- 元音：
  - `a i u e → a i u e`
  - 长短不保留
- 词形：
  - 词尾辅音补 `a`

#### 5.9 古埃及语

古埃及语一律依本节采用的埃及学传统读音处理。

- 辅音：
  - `ḫ ẖ ḥ → h`
  - `q → k`
  - `ꜥ` 一律省略
  - `ỉ → i`
- 元音：
  - 非重建部分按传统读法落入 `a i u e o`
- 词形：
  - 词尾辅音补 `a`
